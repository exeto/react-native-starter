# react-native-starter

> React Native Starter

## Install

```bash
$ git clone https://github.com/exeto/react-native-starter.git
$ cd react-native-starter && npm i
```

## Usage

```bash
# iOS
$ npm run start:ios

# Android
$ npm run start:android
```
